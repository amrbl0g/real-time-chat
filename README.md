# Real-time Chat

Real-time Chat is a modern messaging app built with **Next.js**, **TypeScript**, **Convex**, **Zod**, **Tailwind CSS**, and **Better-auth**.  
It focuses on fast, real-time conversations with a clean UI, type-safe APIs, and secure authentication.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Backend / Realtime**: Convex
- **Validation**: Zod
- **Styling**: Tailwind CSS v4
- **Authentication**: Better-auth and `@convex-dev/better-auth`
- **Tooling**: Biome (linting and formatting)

## Features

- **Real-time messaging** between authenticated users
- **Secure authentication** using Better-auth
- **Type-safe server and client code** powered by TypeScript and Zod
- **Modern UI** with Tailwind CSS
- **Production-ready Next.js setup** with build, start, lint, and format scripts
